# Телеграм бот для анонимных сообщений
- Простая реализация с своебраной фикцией отправки запросов в DB(база данных) и Cache (кэш)
    - Для создания бота использался фреймфорк telebot, asyncio на Python
    - Для создания фикции отправки сообщений в DB и Cache использовался socket, queue, thread.
        - Мультипроцессорность и асинхронность не использовались
    - Ключ бота валиден (в чём каюсь)
# Примечания
- Только для обучения
