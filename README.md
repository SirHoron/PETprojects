# 📚 Learning Projects Collection

Добро пожаловать в коллекцию учебных проектов.  
Этот репозиторий объединяет небольшие самостоятельные проекты, каждый из которых создан для отработки конкретных навыков, технологий или подходов в разработке.

Цель — не просто написать код, а закрепить теорию на практике, поэкспериментировать с разными инструментами и иметь под рукой работающие примеры для повторения.

---

## 🎯 Для кого этот репозиторий

- Начинающие разработчики, которые хотят видеть реальные примеры кода.
- Студенты, желающие дополнить теорию практикой.
- Все, кто ищет идеи для небольших pet‑проектов.
- Преподаватели, которым нужны готовые примеры для занятий.
